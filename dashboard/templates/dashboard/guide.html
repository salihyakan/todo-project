Ked-O Kullanım Kılavuzu

🐱 Ked-O'ya Hoş Geldiniz!

Ked-O, üretkenliğinizi artırmak için tasarlanmış kapsamlı bir görev yönetim platformudur. Bu kılavuz, tüm özellikleri etkili bir şekilde kullanmanıza yardımcı olacaktır.

İpucu: Platformu keşfederken bu kılavuzu açık tutun ve adım adım ilerleyin.

🚀 Hızlı Başlangıç

1. Hesap Oluşturun

30 saniyede ücretsiz hesabınızı oluşturun

2. İlk Görevinizi Ekleyin

Görevler sayfasından başlayın

3. Takviminizi Planlayın

Etkinliklerinizi takvime ekleyin

4. İlerlemenizi Takip Edin

Analizlerle performansınızı görün

🎯 Temel Özellikler

Görev Yönetimi

Görevlerinizi oluşturun, organize edin ve takip edin.

  • Öncelik seviyeleri (Düşük, Orta, Yüksek)
  • Kategori desteği
  • Son tarih takibi
  • Durum yönetimi (Yapılacak, Devam Ediyor, Tamamlandı)
  • Detaylı görev açıklamaları
Görevlere Git Kategoriler
Not Alma

Zengin metin editörü ile detaylı notlar oluşturun.

  • Zengin metin formatlama
  • Kategori organizasyonu
  • Görev ilişkilendirme
  • Sabitleme özelliği
  • Hızlı arama ve filtreleme
Notlara Git
Takvim Entegrasyonu

Etkinliklerinizi görsel takvimde planlayın.

  • Görsel planlama arayüzü
  • Otomatik hatırlatıcılar
  • Renk kodlu etkinlikler
  • Hızlı etkinlik ekleme
  • Günlük/haftalık/aylık görünüm
Takvime Git
Pomodoro Zamanlayıcı

Odaklanma sürelerinizi verimli yönetin.

  • 25/5 dakika döngüsü
  • Ayarlanabilir zaman aralıkları
  • Oturum takibi
  • Verimlilik analizi
  • Sesli bildirimler
Pomodoro'ya Git
